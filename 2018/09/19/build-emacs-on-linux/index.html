Build Emacs on Linux

2018-09-19
1 minute read
Software > emacs

Build Emacs 26.1 on Ubuntu:

sudo apt-get update
sudo apt-get install build-essential
sudo apt-get build-dep emacs24
cd emacs-26.1
./configure --prefix=/usr/local/
sudo make -j4
sudo make install
